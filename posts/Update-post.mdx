 ## REST HTTP Methods Cheatsheet
GET > Requests data from a specified resource.

POST > Sends data to a server to create or update a resource.

DELETE > Deletes a resource.

PUT > Sends data to a server to create or update




OPTIONS > Describes the communication options for a target resource.

HEAD > Similar to GET but without a response body.

CONNECT > Creates a tunnel to the server.
